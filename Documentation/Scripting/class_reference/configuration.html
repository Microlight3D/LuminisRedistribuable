Configuration

Description

The configuration class is composed of a list of Parameters and ways to access it. Two configuration objects are available in the scripting API: SlicingConfiguration and HardwareConfiguration. HardwareConfiguration contains all hardware related parameters whereas SlicingConfiguration contains slicing related configuration such as resist and objective definitions.

Property Descriptions

Type : str

Configuration type. Should be “micro-FAB3d”.

Parameters : Parameter[]

List of configuration Parameter.

Method Descriptions

hasHardware ( hardware : str ) -> bool

Returns True if the configuration has parameters assigned to the given hardware, False otherwise.

findSpecification ( hardware : str, name : str ) -> Parameter

Returns the first Parameter matching the given name and hardware or None if none is found.

findAllSpecifications ( hardware : str ) -> Parameter[]

Returns a list of all Parameter assigned to the given hardware
