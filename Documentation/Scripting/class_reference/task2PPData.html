Task2PPData

ML3DFab.Models.FabricationTask.Task2PPData

Description

Task2PPData are a part of every fabrication task they contain all the raw data defining the task’s printing trajectories, laser power, speed, etc. There is one Task2PPData in each fabrication task.

getVertices ( ) -> float[]

Get the array of all slicing segments. The vertices array can be sorted in an arbitrary way but alway keeping path integrity. An array of float, representing 3 by 3 the fabrication points.

getPathIndices ( ) -> int[]
