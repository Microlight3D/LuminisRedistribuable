Transform

Interface.Common.Models.Polyhedrons.Transform

Description

A transform stores the translation P, orientation axis X,Y,Z and scale S of an orthogonal coordinate system in 3D space as a 4x4 matrix like so :

\[\begin{split}T = \begin{bmatrix} {X}_{x} \cdot {S}_{x} & {Y}_{x} & {Z}_{x} & {P}_{x} \\ {X}_{y} & {Y}_{y} \cdot {S}_{y} & {Z}_{y} & {P}_{y} \\ {X}_{z} & {Y}_{z} & {Z}_{z} \cdot {S}_{z} & {P}_{z} \\ 0 & 0 & 0 & 1 \\ \end{bmatrix}\end{split}\]

This format allows to effortlessly transform a vector in 3D space to the defined coordinate system. The class itself contains plenty of helper functions to access, modify and apply the transform matrix.

getTranslation ( ) -> Vector3

Returns the transform’s translation P.

getScale ( ) -> Vector3

Returns the transform’s scale S.

getRotationEulerXYZ ( ) -> Vector3

Returns the transform’s rotation in XYZ Euler angles (in degrees) form

getX ( ) -> Vector3

Returns the X axis of the transform’s coordinate system.

getY ( ) -> Vector3

Returns the :ref:Y axis<class_transform_matrix>` of the transform’s coordinate system.

getZ ( ) -> Vector3

Returns the Z axis of the transform’s coordinate system.

getMatrix ( ) -> Matrix4x4

Returns the transform’s matrix.

setPosition ( position : Vector3 ) -> None

Sets the transform’s translation P.

setScale ( scale : Vector3 ) -> None

Sets the transform’s scale S.

setRotationEulerXYZ ( rotation : Vector3 ) -> None

Set the X, Y and Z axis of the transform to mirror the given XYZ Euler angles in degrees.

setMatrix ( matrix : Matrix4x4 ) -> None

Set the transform’s matrix.

applyTo ( vector : Vector3 ) -> Vector3

Return the given vector transformed by the transform matrix.
