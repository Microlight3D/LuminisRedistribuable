Fabrication

ML3DFab.Models.FabricationTask.Fabrication

Description

A Fabrication is the container for all print data including fabrication tasks, print order etc. There is a single Fabrication object per Luminis instance. This object is available in python scripts through the fabrication object. Direct modifications to this object allows to add, remove or modify fabrication tasks and modify print order.

Property Descriptions

Configurations : Configuration[]

Hardware configuration required to print every tasks in the fabrication. This is optional and not used in 2PP fabrication handling yet.

Assemblies : Assembly[]

Unsorted list of fabrication assemblies. Assemblies contain fabrication tasks.

Blueprint : Task[]

List of all fabrication tasks ordered by print order. This list defines what tasks will be printed. If a task is not present in the Blueprint list it will not be printed.

Method Descriptions

findAssemblyById ( id : str ) -> Assembly

Returns the first assembly matching the given id in Assemblies if it exists, null otherwise.

findTaskByIdFromAssemblies ( id : str ) -> Task

Returns the first task matching the given id in all tasks of all assemblies in Assemblies if it exists, null otherwise

findTaskByFileName ( fileName : str ) -> Task

Returns the first task matching the given file name in Assemblies if it exists, null otherwise.

findTaskByIdFromBlueprint ( id : str ) -> Task

Returns the first task matching the given id in Blueprint if it exists, null otherwise.

containsId ( id : str ) -> bool

Returns true if a task with the given id is present in either any assembly of Assemblies or in Blueprint, false otherwise.

addAssembly ( assembly : Assembly ) -> None

Add an assembly to Assemblies. Fires the AssemblyAdded event. If the assembly contains tasks also fires the TaskAddedToAssembly event for each of its tasks.

removeAssemblies ( assemblyArray : Assembly[] ) -> None

Remove the given assembly from Assemblies. Fires the AssemblyDeleted event. If the assembly contains tasks also fires the TaskDeletedFromAssembly event for each of its tasks

removeTask ( task : Task ) -> None

Remove remove the given task from Assemblies and Blueprint.

removeTasks ( tasks : Task[] ) -> None

Remove all tasks in the given “task” array from Assemblies and Blueprint.

addTasksToBlueprint ( tasks : Task[] ) -> None

Add the given tasks in the given “task” array to Blueprint only.

addTaskToBlueprint ( task : Task ) -> None

Add the given task to Blueprint only.

clear ( ) -> None

Remove all assemblies from Assemblies and tasks from Blueprint
