.. _class_configuration:

Configuration
=============

Description
-----------

The configuration class is composed of a list of :ref:`Parameters<class_parameter>` and ways to
access it. Two configuration objects are available in the scripting API: SlicingConfiguration and 
HardwareConfiguration. HardwareConfiguration contains all hardware related parameters whereas 
SlicingConfiguration contains slicing related configuration such as resist and objective 
definitions.

Method Descriptions
-------------------

.. _class_configuration_method_hasHardware:

**hasHardware** **(** hardware : `str`_ **)** -> `bool`_

Returns True if the configuration has parameters assigned to 
the given hardware, False otherwise.

----

.. _class_configuration_method_findSpecification:

**findSpecification** **(** hardware : `str`_, name : `str`_ **)** -> :ref:`Parameter<class_parameter>`

Returns the first :ref:`Parameter<class_parameter>` matching the 
given name and hardware or `None`_ if none is found.

----

.. _class_configuration_method_findAllSpecifications:

**findAllSpecifications** **(** hardware : `str`_ **)** -> :ref:`Parameter<class_parameter>`\[\]

Returns a list of all :ref:`Parameter<class_parameter>` assigned to the given hardware
