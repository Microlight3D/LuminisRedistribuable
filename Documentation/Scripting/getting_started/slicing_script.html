Slicing via script

In this exemple we will write a script that loads an object from the file system and slice it in all the qualities options. Here is the full code bellow:

 1from Interface.Common.Models import SlicingParameters, Quality
 2from ML3DFab.Models.FabricationTask import Task2PP, Assembly2PP
 3from Interface.Common.Models.Polyhedrons import Polyhedron
 4from Interface.Utils.FileImporters import FileImporter
 5from System.Numerics import Vector3
 6from ML3DSlicer import MeshSlicer
 7from System import Single, UInt32
 8
 9LOADPATH = ""
10SPACING = 5
11
12objective = next(obj for obj in SlicingParameters.Objectives if "20X/0.8" in obj.Name)
13resist = next(r for r in SlicingParameters.Resists if "OrmoComp" == r.Name)
14slicer = MeshSlicer()
15
16polyhedron : Polyhedron = FileImporter.Load(LOADPATH)
17vertexArray : Array[Single] = polyhedron.Mesh.getSurfaceVertexArray(0)
18indexArray : Array[UInt32] = polyhedron.Mesh.getSurfaceIndexArray(0)
19normalArray : Array[Single] = polyhedron.Mesh.getSurfaceNormalArray(0)
20
21qualities = [Quality.HIGHEST, Quality.HIGH, Quality.STANDARD, Quality.LOW, Quality.LOWEST]
22for id, quality in enumerate(qualities):
23
24    SlicingParameters.setSlicingParameters(slicer, objective, resist, quality)
25
26    slicer.setMeshVertices(vertexArray)
27    slicer.setMeshIndices(indexArray)
28    slicer.setMeshNormals(normalArray)
29    slicer.setMeshTransform(polyhedron.Transform.getMatrix())
30
31    slicer.initializeSlicing()
32    slicer.initializeHatching(0,0)
33    while(not slicer.isComputeDone()):
34        slicer.progressComputing()
35
36    task2PP = Task2PP()
37    task2PP.Name = polyhedron.Name+"_"+str(quality)+"_"+str(objective.Aperture)+"NA_"+str(resist.RefractiveIndex)+"RI"
38    task2PP.Data = slicer.getResult()
39    SlicingParameters.setTaskParameter(task2PP, objective, resist, quality)
40    task2PP.Parameters.Position = Vector3(1,1,0)*(slicer.getAABBMax() - slicer.getAABBMin())/2
41    task2PP.Parameters.Position += Vector3(0,(task2PP.Parameters.Position.Y+SPACING)*2*id,0)
42
43    if(fabrication.Assemblies.Count <= 0):
44        fabrication.addAssembly(Assembly2PP())
45    fabrication.Assemblies[0].add(task2PP)
46    fabrication.addTaskToBlueprint(task2PP)

Load a polyhedron from file

First let’s import an object from file using the FileImporter static class:

4LOADPATH = ""

We retrieve a Polyhedron object of which we extract it’s arrays for later use in the slicer:

5indexArray : Array[UInt32] = polyhedron.Mesh.getSurfaceIndexArray(0)
6normalArray : Array[Single] = polyhedron.Mesh.getSurfaceNormalArray(0)

Initialize and run the slicer

Now that we have an object to slice, let’s create and initialize the slicer:

1from ML3DSlicer import MeshSlicer
2
3slicer = MeshSlicer()

The slicer needs informations about voxel size and overlap. For this we can use the SlicingParameters static helper class. We’ll just need an objective, a resist and define a print quality:

1from Interface.Common.Models import SlicingParameters, Quality
2
3objective = next(obj for obj in SlicingParameters.Objectives if "20X/0.8" in obj.Name)
4resist = next(r for r in SlicingParameters.Resists if "OrmoComp" == r.Name)
5
6SlicingParameters.setSlicingParameters(slicer, objective, resist, Quality.STANDARD)

We’ll pass our Polyhedron arrays and transform to the slicer:

1slicer.setMeshVertices(vertexArray)
2slicer.setMeshIndices(indexArray)
3slicer.setMeshNormals(normalArray)
4slicer.setMeshTransform(polyhedron.Transform.getMatrix())

The slicer is now ready to be initialized. All slicing modules need to be initialized separately using their initialization function:

  • initializeSlicing for slicing trajectories;

  • initializeHatching for hatching trajectories;

  • initializeFilling for infill trajectories;

  • initializeTextToPath for text generation;

1slicer.initializeSlicing()
2slicer.initializeHatching(0,0)

To operate the slicing, we call progressComputing over and over while the slicer doesn’t notify the process being done via isComputeDone:

1while(not slicer.isComputeDone()):
2    slicer.progressComputing()

Create the fabrication task

One the slicing process over, we can retreive the result: a Task2PPParameters object. We’ll need to create a task, assign it parameters and set it data to the slicing result:

 1from ML3DFab.Models.FabricationTask import Task2PP, Assembly2PP
 2from System.Numerics import Vector3
 3
 4task2PP = Task2PP()
 5# Give the task a comprehensive name including its resist refractive index, objective apreture and print quality
 6task2PP.Name = polyhedron.Name+"_"+str(Quality.STANDARD)+"_"+str(objective.Aperture)+"NA_"+str(resist.RefractiveIndex)+"RI"
 7task2PP.Data = slicer.getResult()
 8# Assign parameters using objectve, resist and print quality
 9SlicingParameters.setTaskParameter(task2PP, objective, resist, Quality.STANDARD)
10# Set the position to be in fabrication stage bounds
11task2PP.Parameters.Position = Vector3(1,1,0)*(slicer.getAABBMax() - slicer.getAABBMin())/2

Finally we’ll add the task to Luminis:

1if(fabrication.Assemblies.Count <= 0):
2    fabrication.addAssembly(Assembly2PP())
3fabrication.Assemblies[0].add(task2PP)
4fabrication.addTaskToBlueprint(task2PP)

Slice for all print qualities

Now that we know how to slice an object let’s do it multiple times for all print quality. To do so we’ll define an array containing all of the print quality:

1qualities = [Quality.HIGHEST, Quality.HIGH, Quality.STANDARD, Quality.LOW, Quality.LOWEST]

Loop over them and slice our object for each quality:
